
# 배열

### 정의, 성질

- 메모리 상 원소를 연속하게 배치한 자료구조
- O(1)에 k번째 원소를 확인하고 변경할 수 있음
- 메모리 상 데이터가 붙어있어 Cache hit rate가 높다.
    - Cach hit rate (적중률)?
        
        캐시메모리가 있는 컴퓨터 시스템은 CPU가 메모리에 접근하기 전, 먼저 캐시 메모리에 원하는 데이터의 존재 여부를 확인. 필요한 데이터가 있는 경우를 적중(hit), 없는 경우를 실패(miss)라고 함. 이때 요청한 데이터를 캐시메모리에서 찾을 확률을 적중률이라고 말함.
        
- 메모리 상 연속한 구간을 잡아야 해 할당에 제약에 걸린다.

임의의 위치에 있는 원소를 확인 / 변경, O(1)

원소를 배열의 끝에 추가, 제거 = O(1)

임의의 위치에 원소를 추가 / 제거 =O(n)

- 중간에 데이터 추가 / 삭제 시, 그 앞 / 뒤에 있는 데이터를 밀어야하기에 시간 복잡도가 김.
